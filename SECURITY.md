# Security Policy

This repository contains static image assets and JSON metadata; it ships no
executable code. Even so, we take the integrity of the collection seriously.

## Reporting a concern

Please open a [private security advisory](https://github.com/DistTopic/padventures-sprites/security/advisories/new)
or a regular [issue](https://github.com/DistTopic/padventures-sprites/issues) for
any of the following:

- A file that appears corrupted, mislabeled, or unexpectedly large.
- Metadata that does not match the asset on disk.
- An intellectual-property concern about a specific asset (see `NOTICE`).

When relevant, include the asset **ID**, its **category**, and the file path.

## Supported content

The `main` branch always reflects the current, supported release of the
collection. Fixes are applied there through the normal pull-request workflow.
